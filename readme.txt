How to run?
-> create a package named cryptography_AES and copy all the files in it

-> Run file named AES then you should enter file direcries of message, cipher text and decrypted text respectively seperated by spaces

-> The cipher text will be stored in ciher_text.txt and decrypted text will be stored in plain.txt
